• <posicao>: Matriz que contém todas as possíveis combinações de vitória.

    • Função <init()>:

    • Inicializa o estado do jogo, preenchendo selecionado com null e exibindo o jogador da vez.
    • Limpa o conteúdo dos botões do tabuleiro.
    • Chama playCPU() para começar o jogo.

    Função <playCPU()>:

    • Aguarda 500ms antes de fazer uma jogada da CPU.
    • Obtém os índices vazios no tabuleiro.
    • Seleciona um índice aleatório entre os índices vazios.
    • Atualiza o botão com a jogada da CPU.
    • Verifica se houve um vencedor ou empate.
    • Alterna o jogador atual.
    • Chama recursivamente a função playCPU() se o jogo não tiver terminado.

    Função <check()>:

    • Verifica se houve um vencedor ou empate após cada jogada.
    • Percorre todas as combinações de vitória e verifica se o jogador atual ganhou.
    • Se um jogador vencer, exibe um alerta e reinicia o jogo.
    • Se não houver vencedor e todos os espaços estiverem preenchidos, exibe um alerta de empate e reinicia o jogo.

    Função <gameOver()>:

    • Verifica se o jogo terminou.
    • É semelhante à função check(), mas retorna true se o jogo terminou e false caso contrário.

    Inicialização do Jogo:

    • Chama init() para começar o jogo.
